## 1. Pengenalan

### Apa itu STC GasX?

**STC GasX** adalah platform analisis biaya Web3 yang membantu Anda membandingkan biaya transaksi blockchain (dengan biaya gas) terhadap metode pembayaran tradisional off-chain (transfer bank, komisi OTA, biaya admin).

### Untuk Siapa STC GasX?

✅ **Peneliti & Mahasiswa** - Untuk skripsi, tesis, atau disertasi  
✅ **Pebisnis** - Untuk analisis ROI dan keputusan adopsi blockchain  
✅ **Trader Crypto** - Untuk optimasi timing transaksi  
✅ **Pengguna Umum** - Untuk memahami biaya blockchain vs tradisional

### Keunggulan Utama

🔥 **17 Modul Analisis** - Dari kalkulator sederhana hingga analisis carbon footprint  
🌍 **Multi-Chain** - Bandingkan 5 blockchain sekaligus  
🇮🇩 **Fokus Regional** - Dukungan Rupiah dan mata uang Asia Tenggara  
📊 **Visualisasi Profesional** - Chart dan grafik akademis  
📄 **Export PDF** - Laporan siap submit

## 3. Panduan Setiap Fitur

### 📊 Tab Dashboard

**Fungsi**: Melihat overview perbandingan biaya on-chain vs off-chain

**Cara Pakai**:
1. Lihat **Market Data Panel** di atas untuk data real-time:
   - Harga ETH (USD & IDR)
   - Gas price (Gwei)
   - Kurs USD-IDR
   
2. Perhatikan **4 Summary Cards**:
   - Rata-rata On-chain Cost
   - Rata-rata Off-chain Cost
   - Hemat per Transaksi
   - Success Rate

3. Sesuaikan **Parameters** di kanan:
   - Base Transaction Amount (default Rp 500.000)
   - Bank Transfer Fee (default Rp 6.500)
   - OTA Commission (default 15%)
   - Admin Fee (default Rp 25.000)

4. Lihat **3 Grafik**:
   - **Bar Chart**: Perbandingan per transaksi
   - **Line Chart**: Tren gas fee vs off-chain
   - **Pie Chart**: Breakdown komposisi biaya

**Tips**: Gunakan dashboard untuk mendapat gambaran cepat!

---

### ⚡ Tab Kalkulator

**Fungsi**: Hitung instant biaya on-chain vs off-chain untuk transaksi spesifik

**Cara Pakai**:
1. Masukkan **Transaction Amount** (contoh: Rp 1.000.000)
2. Pilih **Transaction Type**:
   - Simple Transfer (paling murah: 21k gas)
   - ERC-20 Token Transfer (65k gas)
   - DEX Swap (150k gas)
   - NFT Mint/Transfer (85k gas)
   - Complex Contract Interaction (200k gas)

3. Klik **Calculate Costs**

4. Lihat hasil:
   - On-chain cost (ETH + USD + IDR)
   - Off-chain cost (Bank + OTA + Admin)
   - Mana yang lebih murah?
   - Berapa hemat?
   - Rekomendasi

**Tips**: Perfect untuk keputusan cepat - "Mending on-chain atau off-chain ya?"

---

### 🔷 Tab Multi-Chain

**Fungsi**: Bandingkan gas fee di 5 blockchain sekaligus

**Cara Pakai**:
1. Lihat **5 Blockchain Cards**:
   - Ethereum (Sepolia testnet)
   - Base (Coinbase's L2)
   - Polygon (Amoy testnet)
   - Arbitrum (Sepolia)
   - Optimism (Sepolia)

2. Perhatikan **ranking** dari termurah ke termahal

3. Lihat **Bar Chart** untuk perbandingan visual

4. Klik **Refresh Prices** untuk update terbaru

**Tips**: Layer 2 (Base, Polygon, Arbitrum, Optimism) biasanya 70-90% lebih murah dari Ethereum!

### 📤 Tab Data

**Fungsi**: Import data transaksi atau pakai sample dataset

**Cara Pakai - Upload CSV**:
1. Klik **Choose File**
2. Pilih CSV dengan format:
   ```
   Timestamp,Tx Hash,Gas Used,Gas Price (Gwei),Fee (ETH),Fee (Rp)
   ```
3. Klik **Upload and Analyze**

**Cara Pakai - Sample Dataset**:
1. Pilih dari **4 Skenario**:
   - 🛒 E-Commerce Checkout
   - ✈️ Travel Booking
   - 👥 P2P Transfer
   - 💼 Mixed Transactions

2. Klik **Muat Dataset**

3. Lihat hasil di tab Dashboard/Trends/Breakdown

**Download CSV**:
- Klik **Download CSV** untuk export hasil analisis

**Tips**: Pakai sample dataset untuk explore tanpa perlu data sendiri!

## 6. Troubleshooting

### ❌ Problem: Gas price tidak update

**Solution**:
1. Check koneksi internet
2. Klik refresh di Multi-Chain tab
3. Hard refresh browser (Ctrl + F5)
4. Clear browser cache

---

### ❌ Problem: Notifikasi tidak muncul

**Solution**:
1. Check browser notification permission
2. Enable di Alerts tab
3. Set threshold yang realistis (5-30 Gwei)
4. Pastikan browser tidak dalam "Do Not Disturb"

---

### ❌ Problem: CSV upload gagal

**Solution**:
1. Check format CSV (harus 6 kolom)
2. Pastikan header row benar
3. Hapus special characters di data
4. Coba sample dataset dulu

---

### ❌ Problem: PDF export tidak jalan

**Solution**:
1. Pastikan ada minimal 1 transaksi
2. Wait 5-10 detik untuk generation
3. Check popup blocker browser
4. Try different browser

---

### ❌ Problem: Chart tidak tampil

**Solution**:
1. Reload page
2. Check console for errors (F12)
3. Disable ad-blocker temporarily
4. Try incognito mode

---

### ❌ Problem: Dark mode tidak switch

**Solution**:
1. Klik sun/moon icon di header
2. Check system theme preference
3. Clear localStorage
4. Restart browser
