## 💻 Option 2: Run Locally

### Prerequisites
- Node.js 18.17+
- npm/yarn/pnpm

### 3 Commands to Run

```bash
# 1. Clone
git clone https://github.com/mrbrightsides/stc-gasx.git
cd stc-gasx

# 2. Install
npm install

# 3. Run
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) - Done! 🎉

## 📊 Understanding the Dashboard

When you first open STC GasX, you'll see:

### Top Section: Market Data
- **ETH Price** (USD & IDR) - Real-time from CoinGecko
- **Gas Price** (Gwei) - Live from Infura
- **Exchange Rate** (USD-IDR) - From Frankfurter API

### Summary Cards
1. **Avg On-Chain Cost** - Blockchain transaction cost
2. **Avg Off-Chain Cost** - Traditional payment cost
3. **Savings Per TX** - How much you save
4. **Success Rate** - Transaction success percentage

### 3 Charts
1. **Bar Chart** - Side-by-side cost comparison
2. **Line Chart** - Gas fee trends vs stable off-chain
3. **Pie Chart** - Cost breakdown (gas, bank, OTA, admin)

---

## 🎛️ Adjusting Parameters

Right side of Dashboard has sliders:

```
Base Transaction Amount: Rp 500,000
  ↑ Increase for high-value transactions
  
Bank Transfer Fee: Rp 6,500
  ↑ Adjust for your bank (e.g., BI-FAST)
  
OTA Commission: 15%
  ↑ Change for different platforms (10-25% typical)
  
Admin Fee: Rp 25,000
  ↑ Set manual processing costs
```

**Pro Tip**: High OTA commission (>15%) makes blockchain very attractive!

## 🆘 Need Help?

### Quick Troubleshooting

**Q: Charts not loading?**  
A: Refresh page (Ctrl+F5), disable ad-blocker

**Q: CSV upload fails?**  
A: Check format matches template, try sample dataset first

**Q: PDF won't download?**  
A: Check popup blocker, wait 5 seconds for generation

**Q: Gas price shows 0.0?**  
A: That's fixed! You should see 8 decimals now (e.g., 15.12345678)
